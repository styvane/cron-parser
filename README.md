## dcron

A cron job expression parser

### Synopsis

A parser which parses a cron string and expands each field to show the times at which it will run.


### Installation

```$ go get -u  github.com/styvane/dcron
```

Alternatively, you can install from source

```
$ git clone https://github.com/styvane/dcron
$ cd dcron
$ make install
```
### Usage

	The parser required one argument: the string to parse.

#### Options

```
  -h, --help      help for dcron
  -v, --version   version for dcron
```
